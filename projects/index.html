BobGoesToJail

BobGoesToJail is a semantic law interpreter for the English translations for the German criminal law. This project got me started with experimenting with NLP using Word2Vec and Stanford Wordnet. The corpus is sourced from: http://www.gesetze-im-internet.de/englisch_stgb/englisch_stgb.html BobGoesToJail developed along with Fabian Göttl and Aaron Kopp.
