gesetzeSearch

Full text search of the German law code powered by Xapian project.

gesetze2corpus

Python scripts to download HTML files of German law (English translations) and convert save them to a database after running through multiple filters for cleaning up the text. The HTML files are sourced from: http://www.gesetze-im-internet.de

BobGoesToJail

BobGoesToJail is a semantic law interpreter for the English translations for the German criminal law. This project got me started with experimenting with NLP using Word2Vec and Stanford Wordnet. BobGoesToJail developed along with Fabian Göttl and Aaron Kopp.
